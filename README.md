# Proyecto de Optimización - AMPL & Python

Este repositorio contiene modelos de optimización implementados en AMPL y scripts en Python para análisis y visualización de resultados.

## Estructura

- `ampl/`: Modelos AMPL (.mod), scripts `.run` y parámetros `.dat`.
- `python/`: Scripts de análisis, automatización y notebooks.
- `data/`: Datos de entrada para AMPL.
- `resultados/`: Salidas de AMPL y gráficos generados.
